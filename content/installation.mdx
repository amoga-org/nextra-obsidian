# Installation

Get started with Nextra 4 by setting up your development environment.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js 20** or later
- **npm**, **yarn**, **pnpm**, or **bun**

## Create a New Project

### Option 1: Manual Setup

1. Create a new Next.js project:

```bash
npx create-next-app@latest my-nextra-docs --typescript --tailwind --eslint --app --use-npm
```

2. Install Nextra 4 dependencies:

```bash npm2yarn
npm install nextra nextra-theme-docs
npm install -D pagefind
```

3. Configure `next.config.mjs`:

```javascript filename="next.config.mjs"
import nextra from "nextra";

const withNextra = nextra({
  // Use content directory convention
});

export default withNextra({
  // Your Next.js config
});
```

### Option 2: Clone This Repository

```bash
git clone https://github.com/your-username/nextra-4-starter.git
cd nextra-4-starter
npm install
```

## Project Structure

Your Nextra 4 project should have this structure:

```
my-nextra-docs/
├── app/
│   ├── layout.tsx           # Root layout with Nextra theme
│   └── [[...mdxPath]]/
│       └── page.jsx         # Catch-all route for MDX content
├── content/
│   ├── _meta.js            # Navigation configuration
│   ├── index.mdx           # Homepage
│   └── *.mdx               # Your documentation pages
├── mdx-components.tsx       # Global MDX components
├── next.config.mjs         # Nextra configuration
└── package.json
```

## Development Server

Start the development server with Turbopack:

```bash
npm run dev
```

Your site will be available at `http://localhost:3000`.

> [!TIP]
> Turbopack provides faster builds and hot reloading compared to Webpack.
